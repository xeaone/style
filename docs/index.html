X Style

A minimalist css framework that prioritizes semantic classless styling with classed grid and utilities.

Powerful theming for light and dark mode using css variables.

Based on a layer concept where each layer must stack either from lighter to darker (light mode) or darker to lighter (dark mode).

Each layer should define css variables --back, --mean, --fore

Accent colors should be used on elements that can be interacted with.

Layer

Layer elements such as body and article can be stacked by defining the following css variables. 

            article {
                --back-lit: var(--back-lit-1);
                --mean-lit: var(--mean-lit-1);
                --fore-lit: var(--fore-lit-1);
            }
