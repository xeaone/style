Concepts

Theme

To define a root theme you can overwrite these css variables.

                @media (prefers-color-scheme: light) {
                    :root,
                    ::backdrop {
                        --hue: 0deg;
                        --sat: 0%;
                        --alp: 1;

                        --lit-0-1: 95%;
                        --lit-0-2: 30%;
                        --lit-0-3: 5%;

                        --lit-1-1: 85%;
                        --lit-1-2: 70%;
                        --lit-1-3: 10%;

                        --color-back-sat: 50%;
                        --color-back-lit: 70%;
                        --color-back-alp: 1;

                        --color-fore-sat: 50%;
                        --color-fore-lit: 10%;
                        --color-fore-alp: 1;

                        color-scheme: light;
                    }
                }
                @media (prefers-color-scheme: dark) {
                    :root,
                    ::backdrop {
                        --hue: 0deg;
                        --sat: 0%;
                        --alp: 1;

                        --lit-0-1: 5%;
                        --lit-0-2: 65%;
                        --lit-0-3: 95%;

                        --lit-1-1: 10%;
                        --lit-1-2: 75%;
                        --lit-1-3: 85%;

                        --color-back-sat: 60%;
                        --color-back-lit: 10%;
                        --color-back-alp: 1;

                        --color-fore-sat: 60%;
                        --color-fore-lit: 80%;
                        --color-fore-alp: 1;

                        color-scheme: dark;
                    }
                }

Layers

Layer elements define css variables and provide (theming, AAA contrast). body and article 

                --back-hue: var(--article-back-hue);
                --back-sat: var(--article-back-sat);
                --back-lit: var(--article-back-lit);
                --back-alp: var(--article-back-alp);

                --mean-hue: var(--article-mean-hue);
                --mean-sat: var(--article-mean-sat);
                --mean-lit: var(--article-mean-lit);
                --mean-alp: var(--article-mean-alp);

                --fore-hue: var(--article-fore-hue);
                --fore-sat: var(--article-fore-sat);
                --fore-lit: var(--article-fore-lit);
                --fore-alp: var(--article-fore-alp);

                --accent-hue: var(--article-accent-hue);
